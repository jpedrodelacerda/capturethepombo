## bandit16 -> 17

A chave para o próximo nível é obtida depois de submeter a chave para o nível atual em uma porta entre 31000 e 32000 no localhost. Para descobrir quais portas estão abertas, usaremos o nmap.

```
$ nmap -p 31000-32000 localhost
```

Portanto, recebemos a chave privada para o próximo nível.
> A chave pode ser salva na sua máquina copiando o conteúdo ou com o netcat.

Logamos então no próximo nivel com:
```
$ chmod 0600 sshkey.private17
$ ssh -i sshkey.private17 bandit17@bandit.labs.overthewire.org -p2220
```

Logando no próximo nível, podemos ver a chave com um simples```$ cat /etc/bandit_pass/bandit17```
